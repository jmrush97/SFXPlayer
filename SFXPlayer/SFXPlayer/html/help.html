SFX Player – Instructions

SFX Player lets you build and run a cue list of audio files for live performance. It can be controlled either from the Windows desktop application (WinForms) or from the web interface (any device on the same network).

Getting Started

  1. Launch SFXPlayer.exe (as Administrator if you want the web interface enabled).
  2. The web interface URL is shown at the bottom of the window (e.g. http://192.168.1.10:3030/). Open this on any phone, tablet, or computer on the same network.
  3. Use File → New to create a blank cue list, or File → Open to load a saved .sfx file.

Supported File Formats

SFX Player can load the following audio file types:

  • MP3 (.mp3)
  • WAV (.wav)
  • AIFF / AIF (.aiff, .aif)
  • WMA (.wma)
  • AAC / M4A (.aac, .m4a)
  • FLAC (.flac)
  • OGG Vorbis (.ogg)

Adding Cues

Using the WinForms app

  1. Click the + Add Cue button in the toolbar, or right-click in the cue list and choose Add Cue.
  2. Drag an audio file from Windows Explorer and drop it onto the new cue row, or click the file-browse button on the cue.
  3. Type a description in the text box on the cue row. Descriptions support Markdown – bold, italic, lists and headings will be rendered in the web interface.
  4. Adjust Volume (0–100) using the volume knob on the right of each cue.

Using the web interface

File management (adding/removing cues, choosing files) must be done in the WinForms application. The web interface is primarily designed for running the show during a performance.

Removing Cues

WinForms

  • Select the cue you want to delete (click on it so the yellow highlight moves to that row).
  • Click the Delete Cue button, or right-click → Delete Cue.
  • A confirmation prompt will appear (this can be turned off in Settings).

Web interface

Deletion is not available from the web interface.

Adding Fades

WinForms

  1. Select the cue you want to add a fade to.
  2. In the Fade section of the selected cue, enter:
    • Fade In – duration in milliseconds for the audio to fade up from silence at the start.
    • Fade Out – duration in milliseconds for the audio to fade to silence at the end.
    • Curve – Linear (constant rate) or Logarithmic (matches human hearing).
  3. Click Set Fade to apply.

Web interface

  1. Open the web interface and navigate to the desired cue using Previous / Next.
  2. In the Fade panel on the right, adjust the Fade In, Fade Out, and Curve fields.
  3. Click Set Fade. The change is sent to the WinForms app immediately.

Cue Descriptions and Markdown

Both the cue description and the cue list description (Show → Cue List Description…) support Markdown formatting. The web interface renders Markdown automatically. In the WinForms app the raw Markdown text is shown.

## Act 1 – Scene 3

Use **bold** for emphasis, _italic_ for notes.

- Forest ambience at low volume
- Fade in 2 s, fade out 3 s

Auto-run Cues

Right-click a cue and tick Auto-run to have the next cue start automatically when this one finishes. You can also set an optional delay (pause) before the next cue starts.

Saving and Loading Show Files

  • Show files are saved as .sfx (XML) files.
  • Use File → Export Show File to bundle the cue list and all audio files into a single .show archive for sharing.
  • Use File → Import Show File to extract a .show archive.

Connected Clients

The IP addresses of all web-interface clients currently connected appear in the status bar at the bottom of the WinForms window. This lets you confirm that your tablet or phone has connected successfully.
